# Микросток — рабочий процесс именования и метаданных

## Общий контекст

Andrey Kotko (Kotkoa) — микросток иллюстратор. Портфолио: векторные цветочные
иллюстрации, паттерны, свадебные приглашения, ботанические элементы, праздничные
карточки. Платформы: Shutterstock, iStock, Adobe Stock.

Основная задача текущего этапа: именование файлов и простановка метаданных
в MicrostockPlus (microstock.plus/myfiles) для загрузки на стоки.

## Структура портфолио (VECTOR)

Тематические папки с файлами:
- **linen invit** — льняные приглашения

- **postal card** — почтовые открытки с цветами

- **gydrag 2023 / gydrag** — гортензия (одиночные элементы, паттерны)
- **citrus / limon** — цитрусовая тематика
- **tee rose** — чайные розы (паттерны, приглашения, элементы)
- **lavender** — лаванда
- **omela 2025** — омела
- **yellow** — жёлтые абстрактные эффекты, новогодние

## Процесс работы в MicrostockPlus

1. Открыть папку на microstock.plus/myfiles
2. Выбрать файл(ы) → редактировать метаданные
3. Заполнить: заголовок, описание, ключевые слова
4. Установить страну релиза (Spain)
5. Открыть модалку "Уточнения ключевых слов" → проставить галочки для iStock
6. Сохранить → перейти к следующему файлу

## Автоматизация через Claude in Chrome

Все действия на странице выполняются через `javascript_tool` MCP.
Важно: все `window`-функции теряются при перезагрузке страницы.

### Установка страны (Spain)

```javascript
const sel = [...document.querySelectorAll('select')]
  .find(s => [...s.options].some(o => o.text === 'Spain'));
if (sel) { sel.value = [...sel.options].find(o => o.text === 'Spain').value; }
```

### Навигация

- Следующий файл: `document.getElementById('toggle-next').click()`
- Предыдущий файл: `document.getElementById('toggle-prev').click()`
- Сохранить метаданные: `document.getElementById('save-metadata').click()`
- Открыть уточнения: `document.querySelector('.edit_keyword_terms').click()`
